option from a set of options.

## Customization

### Supported Classes

- `flexy-radio`

### Supported Tokens

- `size`
- `dot-size`
- `dot-color`
- `unselected-outline-color`
- `unselected-outline-width`
- `selected-outline-width`
- `selected-outline-color`
- `state-transition-duration`
- `state-transition-easing`
